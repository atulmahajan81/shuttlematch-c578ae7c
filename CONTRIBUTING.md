# Contributing to ShuttleMatch

Thank you for considering contributing to ShuttleMatch! We welcome and appreciate your help.

## How to Contribute

1. **Fork the Repository**: Use the GitHub interface to fork the repo.

2. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/shuttlematch.git
   cd shuttlematch
   ```

3. **Create a Branch**: Create a new branch for your feature or bugfix.
   ```bash
   git checkout -b feature/my-feature
   ```

4. **Make Changes**: Implement your changes and commit them.

5. **Push to Branch**: Push your changes to your fork.
   ```bash
   git push origin feature/my-feature
   ```

6. **Submit a Pull Request**: Use GitHub to create a pull request.

## Code Style

- Use `PEP8` for Python code.
- Use `ESLint` and `Prettier` for JavaScript/TypeScript code.

## Testing Requirements

- Ensure all tests pass before submitting a PR.
- Write new tests for new features or bug fixes.

## PR Process

- Ensure your PR passes all CI checks.
- Provide a clear description of the changes.
